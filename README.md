# Описание
Простое приложение коммандной строки для просмотра аниме с анилибрии. Работает на Linux и Android(через Termux).

## Установка
Скачать последний релиз [Тут](https://github.com/KiberBomzh/Anilibria/releases/latest)

В папке с файлом .whl вписать:
```
pip install "название файла.whl"
```

Или, если через uv, то можно ничего не скачивать, а просто ввести:
```
uv tool install git+https://github.com/KiberBomzh/Anilibria
```

## Использование
Запускать коммандой:
```
libria
```
При запуске без параметроа выводит 5 последних тайтлов на сайте.

### Параметры
```
-s, --search = Запрос для поиска
```

```
-e, --episode = Номер эпизода*
```
*Чтоб не выбирать потом какой нужен эпизод

```
-r, --results = Количество результатов поиска
```

```
-q, --quality = [480, 720, 1080]
```

```
-p, --player = [mx, mpv]
```
Полезно только для Termux и то в редких случаях. По умолчанию выбирается автоматически. Если пк - mpv, если android - mx (MX Player)

```
-o, --output = Путь для загрузки
```
По умолчанию для пк: ~/Anilibria

В Termux нужно перед этим ввести `termux-setup-storage` .

Путь по умолчанию для андроид: /emulated/0/Movies/Anilibria

### Флаги
```
-d, --download
```
Загрузка вместо запуска плеера

```
-t, --torrent
```
Загрузка торрента (флаг -d не нужен)

Использует aria2

```
-m, --magnet
```
Если нужно вместо загрузки получить magnet ссылку

```
-l, --latest
```
Открыть последний тайтл.

Выводит серии последнего просмотренного тайтла. Данные берет с кеша ~/.cache/Anilibria

```
-f, --fast
```
Запустить последнюю просмотренную серию.

Опять же, данные берет с кеша
